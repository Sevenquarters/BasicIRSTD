## 2.1 红外小目标检测研究现状

红外小目标检测（Infrared Small Target Detection, IRSTD）旨在从复杂红外背景中准确识别尺寸极小、纹理极弱且对比度较低的目标。与常规目标检测任务相比，红外小目标通常仅占据极少像素，缺乏稳定的形状、纹理和语义信息，因此极易受到复杂背景、噪声干扰以及成像条件变化的影响。正因如此，IRSTD 长期以来一直是红外图像理解与目标感知领域中的核心问题之一，并在红外预警、空天探测、无人机监测与夜间安防等场景中具有重要应用价值。

从方法演化角度看，IRSTD 的发展大体经历了传统先验驱动、深度特征建模以及统一范式探索三个阶段。早期方法主要依赖局部对比、低秩稀疏分解、张量分解以及显著性建模等显式先验，其基本思想是利用小目标与背景在局部结构、亮度分布或低秩属性上的差异完成目标增强与背景抑制。这类方法通常具有较强的物理可解释性，对大规模标注数据依赖较低，但在复杂背景、弱信噪比和跨场景泛化条件下往往存在性能瓶颈。

在传统方法中，Attentional Local Contrast Networks（ALCNet）是一个具有代表性的过渡性工作。该方法将局部对比先验显式嵌入深度网络结构之中，在保留传统方法解释性的同时增强了端到端学习能力。除此之外，基于各向同性约束的方法尝试利用目标局部几何一致性抑制复杂背景中的伪亮点；基于非凸低秩近似与张量分解的方法则从背景低秩性与目标稀疏性的角度出发，对复杂红外场景中的目标-背景分离问题进行建模。总体而言，传统路线虽然在统一 benchmark 上逐渐被深度学习方法超越，但其在可解释性和先验建模方面仍具有重要参考价值，并且对后续 deep unfolding 与模型驱动网络的发展产生了直接影响。

随着公开数据集逐步成熟，基于深度学习的 IRSTD 方法逐渐成为研究主流。该阶段的大量工作以 U-Net 或其变体为基础框架，通过多尺度特征交互、注意力机制和细粒度结构增强提高弱小目标的检测性能。Dense Nested Attention Network（DNANet）通过密集嵌套交互和级联注意力机制强化多尺度特征流动，是近年来使用最广泛的高性能基线之一。UIU-Net 则通过 “U-Net in U-Net” 的嵌套结构进一步增强多尺度细节恢复能力，在多个 benchmark 上表现出较强竞争力。ISNet 将形状先验显式纳入网络设计，强调红外小目标的几何一致性；RDIAN 从方向感知与感受野增强的角度提升 dim target 在复杂背景中的可分性；ISTDU-Net 则代表了轻量化 U-Net 变体在性能与复杂度折中方面的典型思路。

值得注意的是，近年来部分研究开始重新强调低层特征的重要性。ILNet 与 HintU 等工作指出，红外小目标最关键的判别信息往往并不来自深层语义，而更多体现在浅层的局部亮度差异、边缘响应与弱纹理结构中。因此，如果网络在设计中一味追求更强语义抽象，反而可能削弱对极小目标的敏感性。这一现象说明，IRSTD 与常规自然图像目标检测之间仍存在明显差异，针对红外小目标成像机理进行专门设计仍然十分必要。

在 CNN 路线逐渐成熟之后，Transformer 开始被引入 IRSTD，以缓解卷积网络在长程依赖和全局上下文建模方面的不足。SCTransNet 是该趋势中的代表性方法，它通过空间-通道交叉 Transformer 模块增强跨尺度特征之间的全局交互能力。与传统卷积结构相比，这类方法更适合处理目标与背景高度相似、局部判别线索不足的复杂场景。此外，也有研究尝试在 DNANet 等强 CNN 基线之上引入 Swin Transformer 或 ACmix 等模块，以形成 “CNN 主体 + Transformer 插件” 的混合结构。这说明 Transformer 在 IRSTD 中的角色并不只是简单替代 CNN，而更像是在已有高性能局部建模能力基础上补充全局关系表达能力。

除纯粹的结构演进外，IRSTD 研究还在逐步向统一建模、弱监督和 foundation model adaptation 等方向拓展。RPCANet 通过 deep unfolding 将 RPCA 优化过程网络化，体现了传统低秩稀疏先验与深度学习框架的融合趋势；单点监督相关方法则试图在降低标注成本的前提下保持可接受的检测性能；foundation-driven IRSTD 与 SPIRIT 一类方法进一步说明，该领域正在尝试将红外小目标检测纳入更通用的大模型适配与统一表征框架之中。由此可见，当前 IRSTD 的研究重点已逐渐从“设计更复杂的检测网络”转向“构建更统一、更可解释且更具跨场景泛化能力的检测范式”。

## 2.2 红外-可见融合研究现状

红外-可见图像融合旨在综合红外图像在目标显著性上的优势与可见光图像在纹理细节上的优势，从而生成同时保留热目标信息与场景结构信息的融合结果。传统融合研究长期更关注图像本身的视觉质量，例如边缘清晰度、对比度、纹理保留与主观可读性。然而，对于小目标检测任务而言，融合图像视觉上更清晰并不必然意味着检测性能更优。因此，如何从“图像增强导向”转向“任务有效性导向”，成为近年来该方向的重要变化。

在较早的深度融合研究中，Interactive Compensatory Attention Adversarial Learning 代表了一类更偏图像质量优化的思路。该方法通过交互补偿注意力与对抗学习机制，增强红外与可见光在显著区域和细节区域上的互补表达。其价值主要体现在对融合图像质量的提升，以及对注意力驱动融合机制的探索。然而，这类方法的局限也较为明显，即若缺乏下游任务约束，仅凭图像质量指标很难充分证明其对小目标检测的直接增益。

进一步地，Decomposition-based and Interference Perception for Infrared and Visible Image Fusion in Complex Scenes 体现了另一条重要路线，即在复杂场景中显式建模干扰成分与模态分解结构。该方法不再仅关注融合图像的主观质量，而是开始考虑融合结果在 segmentation、detection、salient object detection 甚至 depth 等多类下游任务中的可用性。这说明融合研究正逐步从单一像素质量优化，转向兼顾复杂场景鲁棒性与任务泛化能力的方向。

Beyond Night Visibility 则进一步强调夜间场景与检测相关性的结合。该方法从多尺度自适应融合角度出发，关注关键目标区域的显著增强，并尝试将融合结果与夜间场景下的目标识别需求联系起来。与单纯面向图像观感的融合研究相比，这类方法更接近真实应用需求，即不仅要让图像“更清晰”，更要让融合结果“更利于检测”。

近年来，任务驱动融合（task-driven fusion）逐渐成为该方向中最值得关注的发展趋势。TSJNet 是这一趋势的代表性方法，它通过引入 target-aware 与 semantic-aware 联合驱动机制，将检测与分割目标显式纳入融合过程。与传统融合先生成图像、再交给下游任务使用的串行范式不同，任务驱动融合更加关注 fused representation 是否真正提升下游任务表现。对以小目标检测为核心任务的研究而言，这一变化具有直接的方法论意义：融合方法的优劣应优先通过下游检测指标而非融合图像的主观观感来评判。

总体而言，红外-可见融合研究正在从“融合结果是否更好看”逐步转向“融合结果是否真正更有助于检测决策”。这一趋势与多模态检测的发展方向高度一致，也为后续将融合模块与检测网络深度耦合提供了重要研究基础。

## 2.3 多模态小目标检测研究现状

与单模态检测相比，多模态小目标检测更加关注不同传感器模态之间的互补性利用问题。RGB、红外与多光谱图像在纹理、热辐射、照明敏感性与目标显著性方面具有不同特点，因此多模态检测理论上能够在复杂环境下获得比单模态更强的鲁棒性。然而，在实际应用中，不同模态的有效性会随着场景条件、目标尺度、遮挡程度和成像质量发生明显变化。因此，多模态检测的核心难点并不只是“是否进行融合”，而在于“如何建立稳定、可信且对环境变化鲁棒的跨模态交互机制”。

在多模态检测研究中，MBNet 是模态不平衡问题建模的代表性工作。该方法指出，不同模态在不同场景下对检测任务的贡献并不对等，若不对这种差异进行显式建模，简单融合反而可能削弱检测效果。Anchor-free Small-scale Multispectral Pedestrian Detection 则进一步说明，小尺度目标在多模态场景下同样能够从 anchor-free 检测范式中受益，这为多模态小目标检测与单阶段检测器结合提供了重要启发。

在更复杂的实际场景中，模态可靠性动态变化问题开始受到更多关注。DroneVehicle 及其相关研究表明，在无人机 RGB-IR 车辆检测场景中，不同模态的可靠性会随着昼夜变化、背景复杂度和视角变化而显著波动，因此 uncertainty-aware learning 成为必要机制。这类方法不再把两种模态视为恒定等权的信息源，而是尝试对不同模态在不同条件下的可信度进行动态建模，从而提升检测鲁棒性。

随着 Transformer 及 cross-attention 机制的发展，多模态检测开始从浅层融合进一步演进为深层交互式融合。ICAFusion 是其中的代表方法，它通过 iterative cross-attention guided feature fusion，在跨模态特征之间建立更充分的显式交互关系。与早期简单拼接或加权融合方法相比，这类方法能够更细致地建模不同模态之间的依赖关系和补偿关系，也更适合处理复杂背景与弱目标并存的场景。由此可见，多模态小目标检测正在逐步形成一条较为清晰的发展主线，即从模态对齐与模态平衡出发，进一步走向 uncertainty-aware interaction 与 cross-attention driven fusion。

此外，随着视觉语言模型和 foundation model adaptation 的发展，多模态小目标检测正在出现新的拓展方向。例如，From Words to Wavelengths 这类工作尝试将 VLM 引入 few-shot multispectral detection，说明该领域已经开始探索更通用的表征迁移与小样本建模能力。虽然这类方法尚处于较新的发展阶段，但其研究意义在于推动多模态检测从专门结构设计进一步走向更统一、更通用的跨模态表征学习框架。

总体而言，多模态小目标检测的研究已经从“多模态信息简单叠加”逐步过渡到“基于模态可靠性建模、全局交互建模与任务驱动优化的深层融合”。这一趋势为后续围绕鲁棒交互机制、轻量化部署以及跨场景泛化展开研究提供了清晰的切入点。

## 2.4 本章小结

综上所述，红外小目标检测已经从传统显式先验驱动逐步发展到 CNN、Transformer 与新范式并行推进的阶段；红外-可见融合则从图像质量优化逐步演化为更强调下游任务有效性的任务驱动融合；多模态小目标检测则从模态对齐与模态平衡问题出发，逐渐转向 uncertainty-aware 与 cross-attention 驱动的深层交互建模。尽管现有方法在检测性能、结构设计和模态互补利用方面取得了明显进展，但在复杂背景抑制、跨场景泛化、统一评测、复杂度透明化以及部署友好性方面仍存在明显不足。因此，围绕更鲁棒、更轻量或更具可解释性的红外小目标检测方法展开研究，仍然具有明确的理论意义与应用价值。
