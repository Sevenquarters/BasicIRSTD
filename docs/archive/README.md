# Archive 说明

本目录用于存放历史过程文档、早期阶段总结、旧版实验报告和工作草稿。

归档原则：

- 这些文件不删除
- 但它们不再作为当前主线入口
- 后续阅读和接手项目时，默认优先看 `docs/README.md` 中列出的主线文档

当前归档分区：

- `project_legacy/`
  - 早期项目阶段总结与旧操作说明
- `experiments_legacy/`
  - 早期实验计划、阶段性报告、已被主线总表吸收的过程文档
- `literature_working/`
  - 综述草稿、阶段性工作稿、写作中间件

如果后续有新文档已经被更高层总结吸收，也建议优先归档，而不是继续堆在主目录。
