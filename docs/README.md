## 3. 目录说明

### `docs/project_tracking/`

只保留当前有效的项目状态入口：

- `CURRENT_STATUS.md`

### `docs/experiments/`

只保留当前主线最有价值的实验总结、设计说明和下一步方向。

### `docs/literature/`

只保留当前有效的综述主报告、论文目录、对比表、BibTeX 和 PDF 目录。

### `docs/archive/`

历史过程文档、早期阶段总结、旧版实验报告、工作草稿统一放这里。

原则：

- 不删除历史资料
- 但默认不让它们干扰当前主线阅读
